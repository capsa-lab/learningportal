## Getting Started

```bash
# Clone the repo
git clone https://github.com/CLFTech/training-hub.git

# Install dependencies
npm i

# Environmental variables
Create your .env.local file in your root directory
Get environmental variables from "Tools and Resources" board.

# Test by running the app
npm run dev
```

## Coding Commands

```bash
# Check out the branches
git branch

# Create a new branch

git checkout new_branch
```
